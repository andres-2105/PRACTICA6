¿Qué es HTML?

El trabajo con HTML5 permite crear sitios web altamente eficientes tanto para los algoritmos de buscadores, HTML actuaría como es esqueleto de la web, definiendo su estructura básica, gracias a las etiquetas, como para los usuarios, mediante el diseño visual de páginas web desarrollado con el lenguaje CSS.

¿Qué es CSS?

CSS añadiría toda la capa de personalización sobre el que la web define su aspecto final. Siguiendo este fundamento resulta muy fácil para un diseñador web realizar cambios en la apariencia de una web sin afectar de manera dramática a su contenido. El contenido siempre será el mismo, solo cambia como aquello que podemos ver

¿Qué es JavaScript?

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, utilizado principalmente en el desarrollo web, pensado para agregar potencial de interacción y dinamismo a las páginas web.
